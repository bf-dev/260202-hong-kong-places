# Hong Kong Places

This is a list of places in Hong Kong.

## Format

When adding places, follow this format:

```
## [Place Name]

- **Address**: [Full address]
- **Google Maps**: [Google Maps link]
- **Notes**: [Any additional notes]
```

Example:

```
## Central Ferry Piers

- **Address**: 9 Man Kwong St, Central, Hong Kong
- **Google Maps**: https://www.google.com/maps/place/Central+Ferry+Piers/
- **Notes**: Ferry to Lamma Island, Cheung Chau, Peng Chau, etc.
```

## Region Naming Convention

Places should be organized by **official/legal regions of Hong Kong** to prevent duplicates or overwriting.

Official districts in Hong Kong (18 districts):
- Central and Western
- Wan Chai
- Eastern
- Southern
- Yau Tsim Mong
- Sham Shui Po
- Kowloon City
- Wong Tai Sin
- Kwun Tong
- Kwai Tsing
- Tsuen Wan
- Tuen Mun
- Yuen Long
- North
- Tai Po
- Sha Tin
- Sai Kung
- Islands

File naming convention: `{region}.md` (e.g., `Central and Western.md`)
